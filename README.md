### FAQ
- Q:为什么通用版不能像PC版那样清屏改颜色?  
- A:这个问题黑米粥工作室也没法解决，因为清屏和改颜色是用Windows的批处理命令实现的，而其他平台上是否也有类似的命令，黑米粥工作室尚未得知。如果大家有什么好的建议，欢迎联系我们。
